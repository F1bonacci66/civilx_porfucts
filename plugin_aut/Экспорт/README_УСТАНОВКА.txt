CivilX Auth Plugin - Инструкция по установке
============================================

Дата создания: 17.09.2025
Версия: 1.0.0

ОПИСАНИЕ
--------
CivilX Auth Plugin - это отдельный плагин авторизации для Revit, который предоставляет:
- Авторизацию и регистрацию пользователей
- Управление профилем пользователя
- Просмотр купленных продуктов
- Активацию/деактивацию продуктов
- Блок состояния подписки

УСТАНОВКА
---------
1. Скопируйте папку "CivilX" в папку AddIns Revit:
   C:\ProgramData\Autodesk\Revit\Addins\2023\

2. Структура должна быть следующей:
   C:\ProgramData\Autodesk\Revit\Addins\2023\CivilX\plugin_aut\
   ├── CivilXAuthPlugin.dll
   ├── CivilXAuthPlugin.pdb
   ├── CivilXAuthPlugin.dll.config
   └── CivilXAuthPlugin.addin

3. Перезапустите Revit

ИСПОЛЬЗОВАНИЕ
-------------
1. При запуске Revit плагин автоматически проверит авторизацию
2. Если пользователь не авторизован, откроется окно входа/регистрации
3. После авторизации откроется профиль пользователя с вкладками:
   - Информация о пользователе
   - Мои продукты
   - Управление профилем
   - Управление аккаунтом

ФУНКЦИИ
-------
- Авторизация через API сайта CivilX
- Регистрация новых пользователей
- Просмотр информации о профиле
- Управление купленными продуктами
- Активация/деактивация продуктов
- Сохранение состояния между сессиями
- Индикатор статуса активации

ТЕХНИЧЕСКИЕ ДЕТАЛИ
------------------
- .NET Framework 4.8
- WPF интерфейс
- API интеграция с http://civilx.ru
- Шифрование токенов авторизации
- Локальное сохранение состояния

ПОДДЕРЖКА
---------
При возникновении проблем проверьте:
1. Правильность установки файлов
2. Наличие интернет-соединения
3. Доступность API сервера
4. Логи в Debug Output
