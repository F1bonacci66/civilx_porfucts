ИНСТРУКЦИЯ ПО УСТАНОВКЕ CIVILX ПЛАГИНОВ
==========================================

ВЕРСИЯ: 06.10.2025 - Добавлена система фильтрации категорий
- Убраны английские названия категорий (Element, View, Family и т.д.)
- Элементы без категории теперь отображаются как "Прочее"
- Добавлен блок "Фильтры" с выбором категорий по 6 блокам
- Новое окно фильтрации с вкладками и чекбоксами
- Валидация выбора категорий при экспорте
- Автоматическая загрузка проектов из папки C:\RevitExporter\Проекты

ПРОБЛЕМА: "сборки CivilX\dataviewer\RevitExporterAddin.dll не существует"

РЕШЕНИЕ: Все файлы должны быть в одной папке!

ШАГИ УСТАНОВКИ:
===============

ВАРИАНТ 1 - Установка в папку CivilX_export (РЕКОМЕНДУЕТСЯ):
1. Оставить все файлы в папке CivilX_export
2. Скопировать ТОЛЬКО файл CivilX.addin в:
   C:\ProgramData\Autodesk\Revit\Addins\2023\
3. В .addin файле уже указаны полные пути к DLL файлам

ВАРИАНТ 2 - Установка в стандартную папку:
1. Скопировать ВСЕ файлы из этой папки в:
   C:\ProgramData\Autodesk\Revit\Addins\2023\
2. Структура должна быть:
   C:\ProgramData\Autodesk\Revit\Addins\2023\
   ├── CivilX.addin                    (манифест)
   ├── CivilX.Shared.dll              (общая библиотека)
   ├── CivilXAuthPlugin.dll           (плагин авторизации)
   ├── RevitExporterAddin.dll         (DataViewer плагин)
   └── main_icon.jpg                  (иконка)

4. Перезапустить Revit

ВАЖНО:
======
- В .addin файле указаны пути без подпапок
- Все DLL файлы должны быть в одной папке с .addin файлом
- CivilX.Shared.dll должен быть доступен для обоих плагинов

СТРУКТУРА ПАПОК ПРОЕКТОВ:
==========================
Плагин автоматически сканирует папку: C:\RevitExporter\Проекты

Ожидаемая структура:
C:\RevitExporter\Проекты\
├── [Название проекта 1]\
│   ├── [Выгрузка 1]\
│   │   ├── [Вкладка 1]\
│   │   └── [Вкладка 2]\
│   └── [Выгрузка 2]\
│       ├── [Вкладка 1]\
│       └── [Вкладка 2]\
└── [Название проекта 2]\
    └── [Выгрузка 1]\
        └── [Вкладка 1]\

Пример:
C:\RevitExporter\Проекты\
├── Жилой дом\
│   ├── Выгрузка 1\
│   │   ├── Вкладка 1\
│   │   └── Вкладка 2\
│   └── Выгрузка 2\
│       └── Вкладка 1\
└── Офисное здание\
    └── Выгрузка 1\
        └── Вкладка 1\

ВАЖНО:
- Плагин автоматически загружает все проекты при запуске
- Если папка C:\RevitExporter\Проекты не существует, создайте её
- Структура папок должна соответствовать описанной выше

ПРОВЕРКА:
=========
После установки в Revit должны появиться:
- Вкладка "CivilX" в ленте
- Кнопка "Data Manager" 
- Плагин авторизации
- При открытии Data Manager должны загрузиться все проекты из папки

Если проблемы остаются:
1. Удалить все файлы из C:\ProgramData\Autodesk\Revit\Addins\2023\
2. Скопировать заново все файлы из этой папки
3. Перезапустить Revit
4. Проверить структуру папок проектов
