# ИНСТРУКЦИЯ ПО УСТАНОВКЕ ДЛЯ REVIT 2021

## Требования
- Autodesk Revit 2021
- Windows 10/11
- .NET Framework 4.8

## Установка

### Шаг 1: Скопируйте файлы
Скопируйте все файлы из папки `Экспорт_2021` в папку дополнений Revit 2021:

**Путь к папке дополнений Revit 2021:**
```
%APPDATA%\Autodesk\Revit\Addins\2021\
```

### Шаг 2: Файлы для копирования
- `RevitExporterAddin.dll` - основная библиотека плагина
- `RevitExporterAddin.pdb` - файл отладки
- `CivilX.Shared.dll` - общая библиотека
- `RevitExporterAddin.addin` - файл конфигурации плагина

### Шаг 3: Перезапуск Revit
После копирования файлов:
1. Закройте Revit 2021 полностью
2. Запустите Revit 2021 заново
3. В интерфейсе должна появиться новая вкладка "CivilX"

## Использование

### Доступ к плагину
1. Откройте проект в Revit 2021
2. Перейдите на вкладку "CivilX"
3. Нажмите кнопку "Data Manager"

### Функции плагина
- Управление проектами экспорта
- Создание вкладок экспорта
- Фильтрация элементов по категориям
- Экспорт данных в CSV формат
- Авторизация пользователей

## Устранение неполадок

### Плагин не появляется в Revit
1. Проверьте правильность пути к папке дополнений
2. Убедитесь, что все файлы скопированы
3. Проверьте права доступа к файлам
4. Перезапустите Revit

### Ошибки при запуске
1. Убедитесь, что установлен .NET Framework 4.8
2. Проверьте совместимость с версией Revit 2021
3. Проверьте логи Revit в папке:
   ```
   %LOCALAPPDATA%\Autodesk\Revit\2021\logs\
   ```
